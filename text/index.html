旅遊分帳怎麼算最公平？AA 制與最少轉帳全攻略

幾個人出去玩，誰先付了什麼、最後誰該還誰多少，常常一筆爛帳。這個工具幫你記錄每筆消費、自動算出每人收支，並用「最少轉帳」把還錢次數降到最低。

為什麼要用「分攤對象」而不是平均

旅遊中不是每筆都全員均分——有人沒參加某餐、有人沒搭某趟車。這工具讓你為每筆消費勾選實際的分攤對象，只有真正享用的人才分擔，比「全部平均」公平得多。

最少轉帳：還錢不必繞圈

5 個人互相有欠款時，若一筆筆還會很多次轉帳。本工具用貪婪演算法計算最少轉帳筆數：把所有人的淨收支算出來後，讓「該付錢的」直接轉給「該收錢的」，總筆數降到最低。

📌 一句話總結：逐筆記錄、勾選實際分攤對象 → 系統算每人收支與最少轉帳 → 大筆由一人先付再分攤 → 用電子支付一次結清。

台電電費怎麼算？累進級距與夏月費率完整解析

電費帳單看不懂？台電住宅電價採「累進級距」，用越多每度越貴，而且夏月（6–9月）費率更高。搞懂級距，才知道怎麼省。

累進級距：用越多越貴

住宅電價分六段，每月用電量越高，超出的部分適用越高的單價。台電帳單通常兩個月一期，計費時會把總度數換算成每月平均，再套用級距。

夏月費率（6/1–9/30）

夏季電價明顯高於非夏月，冷氣是用電大宗。把冷氣設在 26–28 度、搭配電扇、定期清洗濾網，能有效降低夏月電費。

📌 一句話總結：電價累進用越多每度越貴 → 夏月費率更高冷氣是大宗 → 控制每月用電在低級距最有效。

電費常見問題

台電電費怎麼計算？
住宅採六段累進電價，用電越多超出部分單價越高。帳單通常兩個月一期，計費時換算成每月平均度數套用級距，夏月（6–9月）費率高於非夏月。
為什麼這裡顯示固定基本費 0 元？
因本工具目前只試算一般住宅「非時間電價」的累進電能費，計算式中不另外加一筆固定基本費，所以顯示 0 元。這不表示所有台電方案都沒有基本費；時間電價、營業用電、契約容量或其他方案可能有不同的基本費、約定容量費或最低計費規則，應依實際電價方案與帳單為準。
裝智慧電表就會多收基本費嗎？
不一定。智慧電表是記錄用電量與用電時段的計量設備，是否有基本費取決於你實際申請的電價方案，而不是單看電表種類。即使已安裝智慧電表，仍可能沿用住宅非時間電價；只有改申請時間電價或其他方案時，才依該方案的費率與計費規則計算。
怎麼降低電費？
控制每月用電量避免衝上高級距、汰換為能效一級電器、減少待機耗電、分散高耗電時段，都能有效降低電費。

台灣油價參考

中油每週四調整油價，95無鉛約30–35元/公升（依國際油價浮動）。一般小客車油耗約10–15L/百公里，休旅車約12–18L，節能車6–9L。

油錢一年花多少？油耗計算與省油實用技巧

每次加油幾百塊感覺還好，但乘以一年的里程，油費常是一筆可觀支出。算清楚每公里成本，才好評估換車或通勤方式。

油費計算公式

油費 = 行駛里程 × 油耗（L/100km）÷ 100 × 油價。油耗數字可參考車輛能耗標示或自行記錄「兩次加滿之間的里程÷加油量」實測，實際油耗常比官方數據高 10–20%。

每公里成本：比較通勤方式的關鍵

把月油費除以月里程，得到每公里成本，方便和大眾運輸、共享車比較。別忘了開車還有停車費、保養、折舊等隱形成本，每公里完整成本往往是油耗成本的 2–3 倍。

📌 一句話總結：油費＝里程×油耗×油價 → 用每公里成本和其他交通方式比較 → 平穩駕駛、正確胎壓、定期保養最省油。

油耗常見問題

油費怎麼計算？
油費＝行駛里程×油耗(L/100km)÷100×油價。可用兩次加滿之間的里程除以加油量實測，通常比官方數據略高。
怎麼比較開車和搭車哪個划算？
把月油費除以月里程算出每公里成本，再和大眾運輸票價比較。別忘了開車還有停車費、保養、折舊等隱形成本。

停月票划算嗎？

時數費率×每日停車時數×每月停車天數 = 每月時數費。若高於月租費，租月票更划算。台北市公有停車場月租約3,000–5,000元。

停車費怎麼算划算？時數計費 vs 月租比較

天天開車上班，停車費可能比油錢還貴。算清楚時數計費和月租哪個划算，一年能差好幾萬。

損益平衡點

關鍵在你每月停幾天、每天停多久。偶爾開車按時計費較彈性；天天通勤、固定停同一處，月租通常更省。把「每月時數計費總額」和「月租金」一比就知道。

📌 一句話總結：比較每月時數計費總額與月租金 → 天天停選月租、偶爾停按時計費 → 公有停車場通常較便宜。

停車費常見問題

時數計費和月租哪個划算？
把月時數計費總額（費率×時數×天數）和月租金相比。天天通勤月租通常較省；偶爾開車按時計費較彈性。
停車費有每日上限嗎？
多數停車場設有每日收費上限，跨日可能重新計費。長時間停車前務必查清上限與跨日計費方式。

綁約還是自購手機划算？

計算方式：綁約總成本＝月租×期數＋補貼價；自購成本＝手機原價＋自選月租×期數。差異即為兩方案的成本落差，差異越大選擇越清楚。

手機綁約划算嗎？綁約 vs 自購手機完整比較

電信業者的「0 元手機」綁約方案看似划算，但攤開總成本，自購手機＋選低資費常常更省。學會算總費用，才不會被話術影響。

比的是「總成本」不是月租

綁約總成本 = 月租 × 綁約月數 ＋ 手機補貼價；自購方案 = 手機原價 ＋ 自選低月租 × 同月數。常見結果是手機越貴、綁越久，自購＋低資費越有利。

📌 一句話總結：比總成本不比月租 → 手機越貴綁越久自購越省 → 綁約有違約金與資費綁定、彈性較低。

手機資費常見問題

手機綁約和自購哪個划算？

綁約解約要付多少違約金？
違約金依合約而定，常與剩餘綁約月數及補貼金額相關。綁約前務必確認解約條款。

折扣怎麼算？打折、折抵、滿額折一次看懂

「8折」「第二件6折」「滿千折百」算法都不同，店家標示也常讓人混淆。搞懂折扣計算，才不會被「假優惠」騙。

台式「打折」和歐美「off」相反

台灣的「8折」是付原價的80%（省20%）；歐美的「20% off」是省20%（付80%）。雖然結果一樣，但數字方向相反，本工具支援多種折扣輸入方式。

📌 一句話總結：台灣「8折」＝付80%、歐美「20% off」＝省20%，方向相反 → 高單價打折省更多、低單價定額折抵划算 → 慎防先抬價再打折的假優惠。

台灣天然氣費率依縣市由各地氣體燃料公司訂定，約22–28元/度（含基本費約100元/月）。一般家庭每月約10–25度，冬季用量較高。桶裝瓦斯（液化石油氣）費率另計，多用於偏遠地區。

台灣瓦斯費怎麼算？天然氣費率完整解析

台灣天然氣費率依縣市不同，每度約22–28元，加上基本費每月約100元。算清楚瓦斯費，能幫助評估是否值得換購節能爐具。

各縣市費率差異

台灣天然氣由各縣市地方瓦斯公司（如台灣中油、欣欣、大台北瓦斯等）供應，費率略有不同，差異約2–4元/度。台北市和高雄市因管線較完整，費率相對較低。

影響瓦斯費的因素

冬季用熱水量、煮食頻率、暖氣使用都影響用氣量。節能爐具雖貴，但用氣效率高，長期可節省瓦斯費。部分縣市有節能補助，可查詢經濟部能源局。

📌 一句話總結：各縣市費率不同（約22–28元/度）→ 含基本費約100元 → 冬季用量大幅增加 → 節能爐具長期可省費用。

瓦斯費常見問題

台灣天然氣和桶裝瓦斯費率一樣嗎？
不一樣。天然氣（管道瓦斯）費率較低，約22–28元/度；桶裝液化石油氣費率較高，主要用於未接管區域。本工具試算天然氣費率。
瓦斯費為什麼冬天特別高？
冬季洗熱水澡時間長、煮湯取暖頻率高，加上部分家庭使用瓦斯暖氣，用氣量大幅增加，導致費用偏高。
如何省瓦斯費？
縮短熱水器設定溫度（建議不超過50°C）、選用節能爐具、適當控制煮食時間，都能有效降低每月用氣量。
