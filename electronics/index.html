歐姆定律說明

V（伏特）= I（安培）× R（歐姆）。功率 P = V × I = I²R = V²/R。電子電路設計最基礎的公式，適用於線性電阻元件。

Q：歐姆定律適用於所有元件嗎？
A：歐姆定律只適用於「線性電阻」（Ohmic conductor）。二極體、LED、電晶體等非線性元件的 V-I 關係不遵從歐姆定律，需用其他模型分析。

LED 限流電阻公式

R = (Vs - Vf) / If。例：5V 電源、3.3V 藍色 LED、10mA → R = (5-3.3)/0.01 = 170Ω，選標準值 180Ω。功率 = (Vs-Vf) × If，建議選用 2 倍以上額定功率的電阻。

RC 時間常數應用

τ = R × C（秒），是 RC 電路的基本特性。廣泛應用於：濾波電路（截止頻率 fc = 1/(2πRC)）、防彈跳電路、A/D 轉換取樣、定時電路。充電至 63.2% 需 1τ，充至 99.3% 需 5τ。

關於 LLC 諧振轉換器設計

LLC 諧振轉換器因其寬負載範圍下的零電壓切換（ZVS）特性，廣泛用於伺服器電源、LED 驅動與車用充電。FHA（First Harmonic Approximation，基本近似法）以基波正弦近似分析增益，是設計初期快速估算的標準方法。設計時通常讓額定工作點落在 fn=1（諧振點）附近以取得最佳效率，並利用低於 fr 的頻段獲得升壓增益。本工具的諧振頻率 fr = 1/(2π√(Lr·Cr))、電感比 Ln = Lm/Lr、品質因數 Q = √(Lr/Cr)/Rac 均採業界通用定義。
