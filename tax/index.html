所得稅年度與扣除額重點

所得稅的「年度」指的是所得發生年度，不是申報年度。113年度所得於114年5月申報；114年度所得於115年5月申報；115年度所得於116年5月申報。若你要估今年薪資稅負，請選115年度；若是在核對今年剛報完的稅，多半是114年度。

  • 房屋租金自113年起改為特別扣除額，每申報戶上限18萬元，且有自住、無自有房屋與排富規定。
  • 幼兒學前扣除：114年度起第1名15萬元，第2名以上每人22.5萬元。
  • 長照特別扣除：114年度起提高為18萬元，但仍有排富條件。

所得稅試算：先把年度與扣除額整理好，結果才會準

所得稅頁的任務不是把所有稅法一次塞給使用者，而是讓使用者先完成試算，再用長文理解「為什麼結果是這樣」。因此本頁把輸入分成基本資料、家庭扣除、進階特別扣除三段，避免一進來就被大量欄位淹沒。

1. 年度要看所得發生年度

113年度代表2024年的所得，通常在114年5月申報；114年度代表2025年的所得，通常在115年5月申報；115年度代表2026年的所得，通常在116年5月申報。若你正在估今年薪資稅負，應選115年度；若你正在核對剛完成的報稅結果，通常是114年度。

2. 先用標準扣除，再決定是否列舉

多數上班族可先使用標準扣除額快速估算；如果你有大量醫療費、捐贈、保險費、購屋借款利息等憑證，再改用列舉扣除比較。列舉不是一定比較省，只有列舉總額高於標準扣除額時才有優勢。

3. 特別扣除額要特別小心資格限制

房屋租金支出、長照、幼兒學前、教育學費與儲蓄投資都有各自條件與上限。房租特別扣除額雖然對租屋族很有幫助，但需符合自住、非營業使用、境內無自有房屋等條件，且有排富限制。本頁會在輸入上限或可能不適用時提出警示。

4. 視覺化重點：看扣除額如何把所得壓低

所得稅不是用總收入直接乘稅率，而是先扣除免稅額、一般扣除額、薪資特別扣除額與其他特別扣除額，再對綜合所得淨額套用累進稅率。瀑布圖可以讓使用者看到每一項扣除額的貢獻，稅階圖則能看出目前淨額落在哪個級距。

所得稅 FAQ

我只知道年薪，其他欄位可以不填嗎？
可以。先填所得年度、年薪與申報身分即可得到快速估算；扶養、列舉與特別扣除額會讓結果更接近實際申報。
房租支出填超過18萬元會怎樣？
本工具會用18萬元作為上限並顯示警示。實際申報還需符合自住、非營業使用、通常境內無自有房屋與排富等條件。
長照扣除額一定可以用嗎？
不是。長照特別扣除額有資格與排富限制，若適用稅率較高或有特定分離課稅情況，可能不能列報。
為什麼選列舉扣除但稅額沒變少？
可能是列舉金額低於標準扣除額，或你填入的列舉金額不足以降低到下一個稅階。列舉需有憑證，並非一定比較省。
115年度的金額什麼時候用？
115年度是2026年所得，通常在116年5月申報時適用。用來預估2026年薪資稅負時可選115年度。

勞健保薪資扣款：不是薪水乘一個比例這麼簡單

勞健保的核心是「級距」。同樣月薪，在勞保、健保、勞退可能對應不同上限與級距，因此頁面必須明確顯示使用的級距版本。

1. 勞保與健保級距不同

勞保投保薪資有上限，健保投保金額級距則延伸得更高。高薪族常看到勞保已到上限，但健保仍繼續往上對應級距。若把兩者混在一起，薪資單檢查會失真。

2. 眷屬會直接影響健保自付額

健保眷屬通常最多計3口，眷屬掛在投保金額較低的一方名下，保費可能較省。本工具會把眷屬數納入健保自付額，並提示眷屬限制。

3. 獎金補充保費要另外看

年終、績效獎金或其他獎勵性給與，若全年累計超過當月投保金額4倍，超過部分可能被扣2.11%補充保費。這不是所得稅，通常也不是隔年報稅多退少補的項目。

4. 視覺化重點：看薪資扣款堆疊

勞健保最適合用薪資單堆疊結構呈現，讓使用者看到健保、勞保、勞退自提、補充保費分別佔多少，而不是只看一個總扣款。

勞健保 FAQ

月薪低於最低工資時可以試算嗎？
可以，但若是全時工作，月薪低於當年度最低工資通常需要檢查。工具會提出警示，提醒使用者確認工時與僱用型態。
為什麼我的勞保級距和健保級距不同？
因為勞保投保薪資與健保投保金額使用不同級距與上限。薪資越高，兩者差異越明顯。
勞退雇主提繳6%會從薪水扣嗎？
不會。雇主提繳6%是雇主額外成本，應進入勞工個人退休金專戶；勞工自願提繳0%到6%才會從薪資扣除。
補充保費和健保一般保費有什麼不同？
一般保費按投保金額與眷屬計算；補充保費則針對獎金、兼職所得、股利、租金等特定所得來源另行計算。
高薪低報會有什麼風險？
短期扣款可能較少，但會影響勞保給付、職災保障與退休相關權益。薪資單上的投保級距應定期核對。

加班費分類不能混在一起算

平日延長工時、休息日、例假日與國定假日有不同法律效果。休息日加班常見為前2小時1又1/3、後續1又2/3；國定假日出勤則是工資加倍。若公司只用一個固定倍率計算，薪資單可能會低估。

加班費計算：先判斷是哪一種日子，再談倍率

加班費最常見錯誤，是把平日、休息日、例假日、國定假日全部用同一種倍率計算。本頁把日子類型拆開，讓薪資單檢查更接近實務。

1. 平日加班有分前2小時與第3小時後

平日延長工時不是所有時數都同一倍率。前2小時與第3小時起的加給不同，若公司把全部加班時數用同一倍率，可能低估加班費。

2. 休息日與例假日不能混用

休息日可以依法安排加班並給付加班費；例假日則原則上不得出勤，只有天災、事變或突發事件等特殊情況才能使勞工出勤，並需給薪與補假。本工具只做金額估算，並會在例假日欄位有數值時提出警示。

3. 國定假日出勤通常是加倍

國定假日出勤通常應加發一日工資，若超過正常工時，還可能另依延長工時規定計算。實務上最好保留排班表、打卡紀錄與薪資明細。

4. 視覺化重點：倍率時間軸比圓餅更適合

加班費不是比例分配問題，而是時數進入不同倍率區間的問題。因此用倍率時間軸或分段圖，比單純圓餅圖更容易看出哪段時數造成金額增加。

加班費 FAQ

加班費時薪基準為什麼常用月薪÷30÷8？
這是月薪制加班費常見的每小時工資換算方式，用來套用平日、休息日或國定假日的加給倍率。
休息日和例假日差在哪？
休息日可依法安排加班並給加班費；例假日原則不得出勤，僅特殊情形例外，且需加倍給薪與補假。
補休一定比加班費划算嗎？
不一定。補休通常是時數換時數，但加班費有倍率加成；是否划算要看個人時間需求與公司制度。
國定假日加班只算2倍嗎？
一般正常工時內出勤會涉及加倍給薪；若超過正常工時，還可能另依延長工時計算，需看實際出勤時數。
公司用固定加班津貼可以嗎？
固定津貼若低於依法逐項計算的加班費，仍可能不足。建議保留出勤紀錄並對照薪資明細。

年終的「扣繳」與「真正稅負」不同

公司發年終時，若單次給付超過扣繳門檻，通常先預扣5%所得稅；但隔年報稅時，年終仍會併入綜合所得總額，用你的邊際稅率計算。高所得者可能補稅，低所得者可能退稅。

年終獎金：入帳金額、預扣稅與最終稅負要分開看

年終獎金不是看到入帳就結束。發放時可能先扣5%所得稅與補充保費，但隔年申報時仍會併入全年所得，用累進稅率結算。

1. 5%扣繳只是先扣，不是最終稅率

公司發年終時若達到扣繳門檻，會先預扣所得稅。隔年申報綜所稅時，這筆年終仍會加進全年所得；如果你的邊際稅率高，可能補稅；如果全年稅負低，可能退稅。

2. 補充保費通常不會退

獎金若全年累計超過投保金額4倍，超過部分可能扣2.11%補充保費。這筆是健保制度，不是所得稅扣繳，和隔年報稅多退少補不同。

3. 分次發放不一定真的省稅

分次發放可能影響單次扣繳與補充保費計算，但所得稅仍按全年所得總額結算。真正要估年終實際稅負，應把不含年終的年薪與年終一起計入。

4. 視覺化重點：年終現金流去向

年終適合用現金流圖呈現：總年終先分成入帳實領、預扣所得稅、補充保費，再估算併入綜所稅後可能增加的真正稅負。

年終獎金 FAQ

年終被扣5%是不是就不用再繳稅？
不是。5%是預扣，隔年申報時仍要併入全年所得結算，可能退稅也可能補稅。
年終一定會扣二代健保嗎？
不一定。通常要看全年獎金累計是否超過當月投保金額4倍，只有超過部分才可能扣2.11%補充保費。
年終分次發比較好嗎？
分次發放可能影響扣繳與補充保費時點，但所得稅仍看全年所得總額，不代表一定省所得稅。
年終要不要放進所得稅頁一起算？
要。如果想知道最終稅負，應把年終併入全年所得試算，而不是只看發放當下的實領。
年終實領和真正實得差在哪？
實領是發放當下扣完預扣與補充保費後的入帳金額；真正實得還要看隔年綜所稅結算後是否補稅或退稅。

174小時與240小時差在哪？

月薪制勞工的加班費基準常用月薪÷30÷8，也就是240小時；但若想評估「每月正常工作時間」的實際小時價值，常用每週40小時換算約174小時。兩者目的不同：前者是法定工資計算基準，後者是薪資談判與工作價值比較。

月薪時薪換算：240小時與174小時代表不同問題

月薪換時薪最容易吵的不是公式，而是你想回答哪個問題。算加班費、談薪資、比較工作機會，應該用不同分母。

1. 240小時適合加班費基準

月薪制加班費常用月薪除以30日再除以8小時，等同以240小時作為換算基準。這是工資計算邏輯，不代表每個月真的工作240小時。

2. 174小時適合比較正常工時價值

每週40小時乘以52週再除以12個月，約為173.3小時，實務上常用174小時。這比較適合用來比較兩份工作、談薪資或評估月薪是否合理。

3. 實際工時能看出隱形低薪

如果工作需要長期無薪加班、假日回訊息或長通勤，名目月薪看起來不低，但實際每小時價值可能下降。這也是未來可再強化的方向。

4. 視覺化重點：雙尺度尺規

月薪時薪不是比例拆解問題，所以不適合只用圓餅圖。雙尺度尺規能同時呈現240小時與174小時的差異，讓使用者知道哪個數字該用在哪裡。

月薪時薪 FAQ

月薪4萬的時薪到底是多少？
若算加班費基準，約為40000÷30÷8＝166.7元；若算正常工時價值，約為40000÷174＝229.9元。兩者用途不同。
談薪水應該用174小時還是240小時？
談薪水或比較工作機會通常用174小時較接近正常工時價值；計算加班費則用加班費基準。
時薪制和月薪制最低工資可以直接比較嗎？
不建議直接比較。時薪制與月薪制的給薪保障、例假日與排班方式不同，應先確認僱用型態。
每週工時超過40小時要注意什麼？
若長期超過40小時，應檢查是否有加班費、補休或變形工時制度，不能只看月薪換算。
通勤時間要不要算入真實時薪？
法律工時通常不含一般通勤，但從個人生活成本角度，通勤會影響真實時間報酬，適合用於工作比較。

薪資單反推適合用來抓異常

如果你只知道實領金額，可以用反推工具估算合理的稅前月薪範圍。若反推結果和合約月薪差距很大，可能是有額外扣款、投保級距異常、所得稅預扣或請假扣款，需要對照薪資單明細。

薪資單反推：用來抓異常，不是取代明細

薪資單反推適合在你只知道實領金額時，先估出合理的稅前月薪範圍。它的價值不是給出唯一答案，而是幫你發現扣款是否可能異常。

1. 先分固定扣款與浮動扣款

勞保、健保、勞退自提通常是固定扣款；所得稅預扣可能因公司制度不同；請假、福利金、工會費、餐費、借支或獎金扣回則屬於浮動項目。若全部混在一起，反推結果就會偏差。

2. 反推結果差很多時，要追薪資明細

如果反推月薪與合約月薪差距大，不一定代表公司錯，但一定代表還有資訊缺漏。這時應請公司提供完整薪資明細，包含薪資項目、扣款項目、投保級距與出勤資料。

3. 投保級距是薪資單檢查重點

高薪低報可能讓短期扣款降低，但會影響勞保給付、職災保障與退休相關權益。反推工具會把級距放進估算，協助使用者發現可能落差。

4. 視覺化重點：扣款率儀表

薪資單反推最需要看的不是比例漂亮不漂亮，而是扣款率是否偏離常態。儀表圖能把扣款率放到低、中、高壓力區間，比圓餅圖更有判斷價值。

薪資單反推 FAQ

反推月薪一定準嗎？
不一定。它是估算工具，若薪資單含津貼、請假扣款、獎金、福利金或其他扣款，結果會有差距。
實領比預期少一定是被多扣嗎？
不一定。可能是所得稅預扣、健保眷屬、勞退自提、請假扣款、福利金或其他制度扣款，應先看明細。
可以用反推判斷高薪低報嗎？
可以提供線索，但不能單獨判定。仍需對照薪資單上的投保薪資、實際薪資與當年度級距表。
沒有薪資明細怎麼辦？
建議先向人資或會計正式索取明細。若長期無法取得，可諮詢勞工主管機關或1955。
反推時要不要把加班費放進實領？
若要估固定月薪，最好排除加班費與獎金；若要檢查當月薪資單，則可把當月所有實領項目一起看。

為什麼信用卡回饋不放在稅務薪資主工具？

信用卡回饋更接近「生活費管理」或「理財工具」，和所得稅、勞健保、加班費的法規屬性不同。建議移到生活費頁或獨立理財工具，避免稅務薪資頁主題分散。

這個主題還可以再強化的方向

稅務薪資頁的核心價值是「法規正確、年度清楚、薪資單看得懂」。除了目前工具外，建議後續可加入更多實務場景，讓使用者從試算走到決策。

1. 加入「年度更新檢查器」

稅務、健保、勞保與最低工資每年都有更新風險。可在頁首加入版本檢查卡：所得稅年度、健保級距、勞保級距、最低工資、補充保費率、加班費法規最後核對日期。若超過一年未更新，頁面自動顯示「請重新核對」。

2. 加入「薪資單異常偵測」

使用者輸入月薪、投保級距、實領金額與扣款明細後，系統可提示：投保薪資疑似低報、健保眷屬可能掛錯、所得稅預扣偏高或偏低、勞退自提與雇提混淆等。

3. 加入「年度總薪酬視角」

很多人只看月薪，但真實薪酬包含年終、分紅、加班費、補助、勞退雇提、保險福利。可新增「總薪酬包」工具，把公司成本、員工實領與退休金提撥分開呈現。

4. 加入「報稅前檢查清單」

依使用者身分產生清單：租屋族檢查房租扣除、育兒家庭檢查幼兒學前、扶養長輩檢查長照與70歲以上免稅額、投資族檢查股利合併或分離課稅。
